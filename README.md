# 2026 社區賦能與專案管理中樞 (Community Empowerment Projects)

歡迎來到 2026 年度專案管理中心！本倉儲（Repository）用於追蹤與協作跨部門的核心專案。
我們透過透明、系統化的方式，確保每一個社區行動都能轉化為具體的社會影響力。

## 🎯 當前核心專案 (Active Projects)

### 1. 🎨 髒髒畫體驗 (4/12)
* **目標對象**：約 30 人（課輔班學童 + 社區孩子）
* **專案成效**：提供高壓或過動兒童感官宣洩管道，促進社區融合。
* **目前狀態**：籌備中 (材料盤點與志工招募)

### 2. 👻 畢業鬼屋：時光的惡作劇 (6月)
* **專案場域**：日式平房木屋
* **專案成效**：透過「沉浸式設計」與「角色扮演」，讓畢業生克服恐懼，並將對未來的焦慮留在木屋中，完成道別儀式。
* **目前狀態**：企劃發想中 (空間動線規劃)

### 3. 🚗 永齡基金會捐車儀式與影響力紀錄 (4-5月)
* **專案任務**：
  1. 兒童畫作數位化與車體塗裝（4-5月）。
  2. 捐贈儀式籌備（時間待定）。
  3. 永續影響力影片拍攝與結案報告撰寫。
* **目前狀態**：視覺設計與腳本聚焦中

## 🤝 跨部門協作指南 (How to Contribute)

如果您是非技術部門同事（如行政、社工、行銷），請透過以下方式與我們協作：
1. **查看進度**：請點擊上方的 `Projects` 標籤，進入專案看板查看最新進度。
2. **提出需求或點子**：請點擊 `Issues` -> `New Issue`，並選擇對應的表單填寫。
